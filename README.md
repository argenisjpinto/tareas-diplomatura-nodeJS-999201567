# Diplomatura Node.JS

🌟 Tareas

Este repositorio reúne todas las tareas desarrolladas durante la Diplomatura en Desarrollo Web con Node.JS, organizadas en carpetas independientes para facilitar la revisión y el seguimiento del progreso. Cada carpeta contiene su propio código fuente y los materiales necesarios para ejecutar o analizar cada ejercicio.

🎯 Objetivo del Repositorio

Centralizar todas las tareas de la diplomatura en un único espacio.

Mantener un orden claro para la revisión docente.

Permitir que cada práctica sea ejecutada de forma independiente.

Documentar el progreso en Node.JS, JavaScript y desarrollo web moderno.

🧩 Contenido de Cada Tarea

Cada carpeta de tarea puede incluir:

Código fuente dentro de /src

Archivos necesarios para su ejecución

Un README.md con instrucciones o detalles del ejercicio (si aplica)
